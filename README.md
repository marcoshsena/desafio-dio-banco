## Desafio Controle de Fluxo

Desafio Criando um Banco Digital com Java e Orientação a Objetos, proposto pelo professor Venilton Falvo Jr.

## Elaboração do Projeto

Seguindo as instruções do desafio, foram feitas as classes para a criação de uma conta corrente e uma conta puopança, onde elas seguem as regras de uma classe pai Conta. Otras classes como Banco e Cliente foram criadas para que possamos obter mais informações ao imprimir as contas na classe Main.
